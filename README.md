*   **SOS Emergency System:** One-tap activation with siren, screen flashing, and simulated alert broadcasting.

*   **Trusted Contacts:** Manage emergency contacts stored in your browser.

*   **Design:** Beautiful Glassmorphism UI with Deep Pink/Purple gradients and Tailwind CSS.
